## Machine-Readable Asset Package

### ScubaBaseline.json

This directory also contains `ScubaBaseline.json`, a machine-readable asset package derived from the markdown baselines. This file is automatically generated when baseline markdown files are modified in pull requests.

**Important**: The markdown files are the authoritative source. The JSON is provided as a convenience for machine consumption only.

### Documentation

- **[Schema Documentation](../../../docs/misc/scubabaselineschema.md)** - Complete JSON structure and field definitions

### Quick Usage

**PowerShell:**
```powershell
$baseline = Get-Content "schema/ScubaBaseline.json" | ConvertFrom-Json
$aadPolicies = $baseline.baselines.aad
```

**Python:**
```python
import json
with open('schema/ScubaBaseline.json') as f:
    baseline = json.load(f)
```

**JavaScript:**
```javascript
const baseline = require('./schema/ScubaBaseline.json');
const products = Object.keys(baseline.baselines);
```

### Getting the Asset Package

The `ScubaBaseline.json` file is available in multiple ways:

1. **GitHub Releases** - Download as a release asset
2. **PowerShell Gallery** - Included with the ScubaGear module
3. **Repository** - Located in this directory after PR merge
